1. Fundamentos de Python

¿Qué es Python? Características y aplicaciones.

Instalación y configuración del entorno: Python, IDEs (VS Code, PyCharm) e IDLE.

Sintaxis básica:

Estructura de un programa en Python.

Variables y tipos de datos.

Entrada y salida de datos (input(), print()).

Operadores básicos (aritméticos, de comparación y lógicos).

Control de flujo:

Condicionales: if, elif, else.

Bucles: while, for, y uso de range().

Control de flujo: break, continue, pass.

Objetivo:

Dominar la sintaxis básica, la lógica de programación y las estructuras de control.

2. Estructuras de Datos y Funciones

Funciones en Python: Definición con def, parámetros y retorno de valores. Funciones integradas y personalizadas. Estructuras de datos: Listas y tuplas: creación, acceso, modificación, métodos (append(), pop(), etc.). Diccionarios y conjuntos: estructuras clave-valor, métodos (get(), keys(), values()). Comprensión de listas (list comprehensions). Manejo de errores: Excepciones con try, except, finally. Objetivo: Aprender a organizar, manipular y procesar datos de forma eficiente.

3. Programación Avanzada con Python

Manejo de archivos: Lectura y escritura de archivos (open(), modos de archivo, bloques with). Módulos y bibliotecas: Importación (import, from ... import) y uso de bibliotecas estándar (os, math, etc.). Programación Orientada a Objetos (POO): Clases, objetos, atributos y métodos. Ejemplos prácticos de POO. Proyectos finales: Integración de conceptos en aplicaciones reales. Objetivo: Ampliar las capacidades del estudiante para desarrollar aplicaciones completas, utilizando conceptos avanzados y buenas prácticas.
