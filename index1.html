Manejo de archivos avanzados

El lenguaje "Python" nos permite leer y escribir archivos usando las funciones open(), read(), write() y close(). De igual foorma Se puede abrir un archivo en diferentes modos, para asegurarse de que el archivo se cierre automáticamente, tambien se puede usar la estructura with.
Las "excepciones" son errores que ocurren durante la ejecución de un programa. En Python, se gestionan usando try, except, else y finally. "try" ejecuta el código que puede generar un error, y "except" lo captura.
Un módulo es un archivo que contiene definiciones y declaraciones de Python y una biblioteca es una colección de módulos.
"OOP" en Python se basa en la creación de clases y objetos, en donde na clase es un plano para crear objetos, y un objeto es una instancia de una clase. Las clases pueden tener características y métodos comportamientos.
